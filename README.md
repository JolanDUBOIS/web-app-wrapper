# web-app-wrapper
Electron wrapper for any website to turn into a desktop app.

Goals:
- Make the app work in Wayland
- Install the app on KDE easily (mostly KDE oriented project)

## Setup

```bash
npm install
```

## Run in development

```bash
npm start
```

## Package

```bash
npm run package
```

## Launch via script

```bash
./scripts/run.sh
```
